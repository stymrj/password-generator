# 🔐 React Password Generator

A simple yet stylish **Password Generator App** built using **React** and **Tailwind CSS**.  
It allows users to create strong and customizable passwords with options for uppercase, lowercase, numbers, and special characters — with one-click copy functionality and toast notifications.

---

## 🚀 Features

✅ Generate secure random passwords  
✅ Customize password length (8–26 characters)  
✅ Include or exclude:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters  
✅ Copy password to clipboard with one click  
✅ Beautiful gradient UI with Tailwind CSS  
✅ Toast notifications using **React Toastify**

---

## 🧩 Technologies Used

| Tool / Library | Purpose |
|-----------------|----------|
| **React** | Frontend framework |
| **React DOM** | React rendering support |
| **React Icons** | Clipboard icons (copy/check) |
| **React Toastify** | Toast notifications |
| **Tailwind CSS** | Styling & responsive layout |
